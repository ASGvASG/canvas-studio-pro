# CanvasStudio Pro

CanvasStudio Pro is a polished multi-platform drawing application built with Flutter. It combines a responsive canvas workspace, layered editing, and professional tool controls for sketching, annotation, and visual ideation.

## Highlights
- Android, Windows, and Linux support
- Layer-based drawing workflows with undo and redo history
- Brush, shape, opacity, and pan/zoom controls
- Release-ready packaging for APK, ZIP, DEB, RPM, and AppImage artifacts

## Quick Start
1. Install Flutter 3.24 or newer.
2. Run `flutter pub get`.
3. Launch the app with `flutter run`.

## Build Targets
- Android: `flutter build apk --release`
- Windows: `flutter build windows --release`
- Linux: `flutter build linux --release --target-platform=linux-x64`
